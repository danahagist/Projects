This is the readme file for a small project using linear regression to predict wine quality.

This is a very useful and simple template that can be used to apply linear regression to comparable problems.

Data Source (via Kaggle and UCI Machine Learning Repository):
P. Cortez, A. Cerdeira, F. Almeida, T. Matos and J. Reis. 
Modeling wine preferences by data mining from physicochemical properties. In Decision Support Systems, Elsevier, 47(4):547-553, 2009.
